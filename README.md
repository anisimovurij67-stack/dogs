# Собачий справочник (Dog API App)

Настольное приложение на C# (Windows Forms), разработанное в рамках прохождения учебной ознакомительной практики. Программа демонстрирует работу с внешними REST API, обработку JSON-данных и принципы асинхронного программирования.

## Функционал

*   **Генерация фактов:** Получение случайных интересных фактов о собаках в виде текста.
*   **Просмотр фотографий:** Загрузка и отображение случайных фотографий собак различных пород.
*   **Асинхронность:** Все сетевые запросы выполняются асинхронно (`async/await`), что предотвращает зависание графического интерфейса во время ожидания ответа от сервера.

## Стек технологий

*   **Язык:** C#
*   **Платформа/Интерфейс:** .NET / Windows Forms
*   **Сетевое взаимодействие:** `HttpClient` (`System.Net.Http`)
*   **Обработка данных:** `JsonDocument` (`System.Text.Json`)
*   **Контроль версий:** Git, GitHub

## Используемые API

Проект интегрирован с двумя публичными открытыми веб-сервисами (не требуют авторизации и API-ключей):
1.  [**Dog API**](https://dogapi.dog/docs/api-v2) — эндпоинт `https://dogapi.dog/api/v2/facts` используется для получения текстовой информации.
2.  [**Dog CEO API**](https://dog.ceo/dog-api/) — эндпоинт `https://dog.ceo/api/breeds/image/random` используется для получения URL-ссылок на изображения.

## Архитектура проекта

Код проекта спроектирован с учетом разделения ответственности (Separation of Concerns):
*   `Form1.cs` — отвечает исключительно за работу с пользовательским интерфейсом (UI), обработку кликов и вывод готовых данных на экран.
*   `DogApiHelper.cs` — выделенный класс бизнес-логики, который инкапсулирует в себе HTTP-клиент, выполняет все сетевые запросы к эндпоинтам, обрабатывает возможные ошибки сети и парсит полученный JSON-ответ.
